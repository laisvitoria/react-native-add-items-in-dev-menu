# react-native-add-items-in-dev-menu

Package to add items in dev menu

## Installation

```sh
npm install react-native-add-items-in-dev-menu
```

## Usage

```js
import { multiply } from "react-native-add-items-in-dev-menu";

// ...

const result = await multiply(3, 7);
```
